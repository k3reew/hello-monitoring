# Система мониторинга простого веб-приложения

## Описание

Решение состоит из простого веб-приложения на Python, которое возвращает строку
`Hello World!`, и скрипта мониторинга, который периодически проверяет доступность
приложения и перезапускает его через `systemd` в случае недоступности.

Периодический запуск мониторинга реализован через `systemd timer`.

## Структура проекта

- `app.py` – веб-приложение, возвращающее `Hello World!`.
- `monitor.py` – скрипт мониторинга:
  - проверяет доступность HTTP-URL;
  - логирует результаты проверки;
  - вызывает `systemctl restart` для перезапуска приложения.
- `config.ini` – конфигурация (URL приложения, имя сервиса, файл лога).
- `helloapp.service` – `systemd`-юнит для веб-приложения.
- `helloapp-monitor.service` – `systemd`-юнит для однократного запуска мониторинга.
- `helloapp-monitor.timer` – `systemd`-таймер, который запускает мониторинг каждые `N` секунд.
- `install.sh` – скрипт установки и обновления.
- `README.md` – данный файл.

## Требования

- Linux с systemd
- Python 3
